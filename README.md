This repository contains a collection of data analysis projects performed in Excel and Statistica. Each project focuses on a different case study, covering various statistical methods and data visualization techniques.

Microsoft Excel: Data cleaning, pivot tables, and core analysis.
Statistica: Advanced statistical testing and modeling.

Statistical Methods Included
The analyses cover both categories of statistical tests:
Parametric Tests: (e.g., t-tests, ANOVA) – used when data follows a normal distribution.
Non-parametric Tests: (e.g., Mann-Whitney U, Kruskal-Wallis) – used for non-normal or ordinal data.
